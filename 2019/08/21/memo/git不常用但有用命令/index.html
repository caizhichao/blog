git不常用但有用命令

删除对某些文件的版本管理

  • git rm -r –cached “bin/“
  • git commit -m” remove bin folder all file out of control”
  • 查看结果没问题了再push

通过命令行创建一个新的远程版本库并推送到远程

  • touch README.md
  • git init
  • git add README.md
  • git commit -m “first commit”
  • git remote add origin ssh://admin@192.168.1.23:29418/qusouti.git
  • git push -u origin master
